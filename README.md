# audio-analyzer
This is a lightweight utility to gets the statistics of a media file. Along with the steaming information, it also checks if the media file is silent or not.

```json
{
    "details": {
        "isSilent": true,
        "name": "no_audio.flv",
        "format": "FLV",
        "frequency": "44100.0 Hz",
        "sampleSizeInBits": "16 bit",
        "frameSize": "4 bytes",
        "type": "little-endian",
        "encoding": "PCM_SIGNED",
        "channelType": "stereo"
    }
}
```
